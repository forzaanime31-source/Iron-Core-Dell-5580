🚀 Iron-Core: Dell Latitude 5580 Extreme Optimization

Questo progetto dimostra come resuscitare un Dell Latitude 5580 eliminando Windows e configurando Debian 13.4.0 per le massime prestazioni.

## 📈 Performance: Windows vs Debian
| Sistema | RAM all'avvio | Esperienza |
| :--- | :--- | :--- |
| Windows 11 | ~3.5 GB | Pesante / Lento |
| **Debian Iron-Core** | **513 MB** | Ultra Leggero / Istantaneo |
